# Project flask end VueJS

Flask api Rest and VueJS.

## Backend

```
git clone https://github.com/rg3915/flask-vuejs.git
cd flask-vuejs
python3 -m venv .venv
cd /backend
pip install -r requirements.txt
python run.py
```

Port : 5000.

## Front-end

```
cd /frontend
npm install
npm run dev
```

Port : 8080.
